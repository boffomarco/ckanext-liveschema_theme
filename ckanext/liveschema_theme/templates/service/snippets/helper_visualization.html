The Visualization service is aimed at providing a visualization toolkit in order to better understand the source schema.
It provides as output a resource containing all the elements of each Class.

In order to visualize the datasets, we rely on the Intervene environment.
Two outputs can be produced, using our resource as input:
  • Knowledge Lotuses, namely the Venn intersections between the main entity types of a schema;
  • Upset plots.
These formats enable us to discover the relations and cross-references existing between different Classes of the dataset.

More information can be obtained here: Intervene documentation.
