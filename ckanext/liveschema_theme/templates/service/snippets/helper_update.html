{{ _('What is the LiveSchema Updater?') }}

The LiveSchema Updater is used to find schema datasets by scraping existing catalogs.
Besides searching and collecting new data, the LiveSchema Updater parses the new resources and produce new reference formats (e.g., TTL, RDF, CSV)

This service can be run by LiveSchema admins.
In this process the already existing catalogs will be updated, then some other new catalogs or single datasets can be added, by manually creating a list with the new source links to be scraped.
