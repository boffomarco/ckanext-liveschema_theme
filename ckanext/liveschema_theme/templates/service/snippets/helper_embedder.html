{{ _('What is a Knowledge Embedder?') }}

This service offers the possibility to generate Models (stored in a .PKL file) and Tensors (stored in multiple sheets .XLSX file) from a given schema dataset.
The output of the Knowledge Embedder is a generalizable context about the input schema that can be used to infer relations and insights about relationships and properties inside the schema.

The Knowledge Embedder implements the PyKEEN package and allows for an easy application and customization of all the related libraries.
For more information about parameters and scope, see the PyKEEN reference page.

For additional information about Knowledge Embeddings techniques, please visit also Introduction to Knowledge Graph Embeddings.
