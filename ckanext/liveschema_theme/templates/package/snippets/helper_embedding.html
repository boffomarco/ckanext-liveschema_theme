{{ _('What is a Knowledge Embedding?') }}

A Knowledge Embedding is composed of a model (stored in a .PKL file) and its tensors (stored in multiple sheets .XLSX file).
The output of the Knowledge Embedder is a generalizable context about the input schema that can be used to infer relations and insights about relationships and properties inside the schema.

For additional information about Knowledge Embeddings techniques, please visit also Introduction to Knowledge Graph Embeddings.
